# 1) O que é software?
Software é um programa de computador e, também, um conjunto de dados e configurações para que tudo nele funcione corretamente. 
Um software é composto por:
- **Instruções de computador**: o algoritmo; devem realizar determinada função quando executadas;
- **Estruturas de dados**: relação com o banco de dados; Como as informações e dados são manipulados;
- **Documentos**: descrição de como o programa funciona, como utilizá-lo.

# 2) Cite três diferenças entre hardware e software.
O hardware é a parte física do computador e software é a parte lógica.

1. O software é desenvolvido, não fabricado como o hardware.
2. Há um padrão sobre as peças do hardware e como ela devem ser constiruídas, sendo mais genérico. Já a maioria dos software é feita sob encomenda -> de forma personalizada para atender necessidades específicas do cliente
3. O hardware dá muito defeito no início da vida, depois se estabiliza, e no fim da vida dá muito defeito de novo. Já o software precisa de manutenção constante.
# 3) O que é engenharia de software?
"A engenharia de software abrange um processo, um conjunto de métodos (práticas) e um leque de ferramentas que possibilitam aos profissionais desenvolverem software de altíssima qualidade." (PRESSMAN, 2011)
Logo, a engenharia de software pode-se dizer, de forma simplificada, é uma ciência que estuda a melhor maneira de desenvolver um software de alta qualidade.

# 4) Para você, quais desafios e facilidades o futuro trará para o desenvolvimento de software?
São desenvolvidas cada vez mais ferramentas para para ajudar no desenvolvimento de software, como editores de texto, frameworks, APIs, etc. O que torna o processo muito mais rápido, sem ter que se preocupar com muitos detalhes.
Mas também tem desafios como: desenvolver sistemas e softwares que permitam se comunicar através da rede.
