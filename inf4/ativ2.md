# Atividade
## O que é processo de software?
   O processo de software é um conjunto de atividades, ações e tarefas que levam à produção de um produto a ser entregue.
   É definido quem faz o quê, quando e como fazer para atingir um determinado objetivo.
  
## Descreva as cinco atividades de uma metodologia genérica do processo de engenharia de software.
1. **Comunicação**: Se comunicar com o cliente; Entender quais são os objetivos do projeto e do que ele precisa.
2. **Planejamento**: Cria-se um *plano de projeto de software*, definindo as tarefas, os riscos, os recursos necessários, qual o produto final e um cronograma de trabalho.
3. **Modelagem**: Cria-se modelos, como diagramas, para representar e entender melhor o software e suas necessidades.
4. **Construção**: Gerar o código e fazer os testes para identificação de erros.
5. **Emprego**: Entrega ao cliente e ele dá um feedback.
