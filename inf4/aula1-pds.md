# O que é software?
"Software de computador é o produto que profissionais de software desenvolvem e dão suporte ao longo do tempo."
O software gearlmente é uma estrutura que manipula os dados.
Um software é constituído de: **instruções, estruturas de dados e documentos**.

## Existem dois tipos de software:
- Genéricos: são criados para qualquer pessoa, como o  office da microsoft.
- Personalizados (ou sob encomenda): são criados de acordo com as especificações do cliente.


## "O software é mais um elemento de sistema lógica do físico."
Num hardware o início é mais fácil de dar defeito. Mas os problemas nas peças vão sendo identificados e são feitos ajustes até que passe muito tempo estável e quando vai chegando no fim da vida útil há muitos problemas.

No software, há duas curvas de desgaste: a idealizada é que tenha muito problema no início, mas depois se estabilize e se mantenha estável. Mas a real é que dá muito problema no início, vai se estabilizando, então alguém pede para mudar a cor do botão e estoura muitos problemas de novo, vai se estabilizando, modifica, aparece novos problemas, vai se estabilizando... Mas com o tempo, o software vai se desgastando por causa do que acontece ao redor, com a falta de atualização de bibliotecas, de configurações de segurança, etc.
