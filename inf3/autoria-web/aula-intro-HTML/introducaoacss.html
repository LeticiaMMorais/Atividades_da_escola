Sintaxe do CSS

Nós temos um seletor para indicar no que eu quero colocar estilo. Para cada seletor, podemos colocar uma ou mais declarações, que são formadas por uma propriedade e um valor.

Importando as CSS

É preciso usar a tag link, onde em rel você indica o tipo de arquivo (no caso é stylesheet) e no href é indicado o caminho para o arquivo css que vai dar estilo à página: 

<link rel="stylesheet" href="css/style.css">

Ele é colocado dentro do head.

Tipos de seletores

Há três tipos de seletores:

  • elemento
  • id
  • classe

Cada um tem uma especificidade. Os valores mais a esquerda na especificidade (ou maior valor) ele vai ter aquela propriedade sobresaída.

Você também pode combinar seletores para especificar mais.

Seletor de ID

O id é único, não pode ser usado o mesmo id para elementos diferentes, ele deve ser usado uma única vez. É como se fosse o CPF do elemtento.

Para indicar um seletor de id, você coloca # seguido pelo o id do elemento.

Tem a maior especificidade -> (1, 0, 0)

Seletor de classe

Posso definir uma mesma classe para mais de um elemento. Eles vão compartilhar caracteríticas comuns, por isso estarão na mesma classe.

Um só elemento pode ter uma ou mais classes, sendo elas separadas por um espaço. Se tiver suas classes e elas alterar uma mesma caracterítica (cor, por exemplo), irá prevalecer a da última classe definida.

Organização

Uma sugestão é organizar o css colocando na ordem:

  1. seletores de elemento
  2. seletores de classe
  3. seletores de id
