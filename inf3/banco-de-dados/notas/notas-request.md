# Notas sobre http request

## CRUD
c ➡️ create ➡️ POST\
r ➡️ read ➡️ GET\
u ➡️ update ➡️ PUT\
d ➡️ delete ➡️ DELETE

---
O cifrão indica que é uma variável e o número indica a posição das variáveis.

A mensagem de UPDATE sendo: "UPDATE 1" significa que foi atualizado 1 valor. 
Sendo assim, o UPDATE pode vir com qualquer número X, dependendo do total de 
atualizações feitas de uma só vez. Se não der certo, ele vai dar erro. Sempre 
que for bem sucedida, será exibida uma mensagem de erro (que não tem UPDATE).

O UPDATE 0 ocorre quando tenta atualizar algo para um valor que ele já contém.


    As APIs feitas no Back, são chamadas no front por meio de JavaScript.
